# Pasture Assessment

## When to Use

Use this skill when interpreting current pasture condition or reconciling conflicting inputs.

## Procedure

1. Gather the latest observations.
2. Distinguish direct field observations from inferred remote signals.
3. Look for forage sufficiency, stress, and recovery patterns.
4. State what remains uncertain.

## Pitfalls

- Do not confuse biomass proxies with exact grass height.
- Do not over-trust imagery when a recent field note contradicts it.
