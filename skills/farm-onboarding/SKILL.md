# Farm Onboarding

## When to Use

Use this skill when a farmer is setting up `openPasture` for the first time.

## Procedure

1. Capture the farm name and timezone.
2. Capture a workable representation of location and boundaries.
3. Add paddocks one by one.
4. Record the first herd.
5. Confirm current paddock assignment if known.
6. Explain what inputs help the agent most.

## Success Criteria

The agent can produce a first morning brief from the information gathered.
