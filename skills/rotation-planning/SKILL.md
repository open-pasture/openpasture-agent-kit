# Rotation Planning

## When to Use

Use this skill when deciding whether animals should move, stay, or wait for more information.

## Procedure

1. Identify the herd and current paddock.
2. Review recent rest periods and observations.
3. Check for practical constraints such as water, labor, and weather.
4. Bring in relevant knowledge entries.
5. Prefer operationally realistic recommendations.

## Heuristics

- Preserve residual.
- Protect recovery.
- Read animals and pasture together.
- Favor simple, explainable decisions over fragile optimization.
