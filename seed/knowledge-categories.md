# Knowledge Categories

Use simple, reusable category names for ancestral knowledge entries.

## Preferred Categories

- `grazing-management`
- `animal-behavior`
- `genetics`
- `infrastructure`
- `forage`
- `soil`
- `water`
- `economics`
- `species`
- `weather`
- `products`

## Rules

- Reuse an existing category before creating a new one.
- Prefer broad, durable terms over narrow synonyms.
- Do not split hairs between near-identical words.
- If `genetics` already exists, do not introduce `breeding` for the same concept.
- If a lesson spans multiple concerns, choose the primary management topic and capture the rest in tags.
