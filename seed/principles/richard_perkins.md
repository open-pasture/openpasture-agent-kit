# Richard Perkins

## Core Themes

- whole-farm design,
- infrastructure realism,
- decision-making discipline,
- context-aware regenerative systems.

## Actionable Lessons

- Use the scale of permanence as a planning lens: climate, landform, water, trees, access, infrastructure, animals, soils.
- Lightweight, modular systems matter because operational flexibility matters.
- The back-end decision process is as important as the visible farm layout.

## Design Implication

The agent should not only choose where animals move. It should preserve enough context that later recommendations remain coherent with the shape of the operation.
