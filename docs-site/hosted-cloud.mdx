# Hosted Cloud

Don't want to self-host? The hosted OpenPasture product wraps the open-source agent
kit with managed infrastructure so you can get started without running anything locally.

<Info>
  The hosted product is in beta. [Join the waitlist at openpasture.com →](https://openpasture.com)
</Info>

## What the hosted product adds

| Capability | Agent Kit (OSS) | Hosted Cloud |
| --- | --- | --- |
| Farm tools, skills, knowledge | Included | Included |
| Local SQLite storage | Included | Managed for you |
| CLI, MCP, Hermes connectors | Included | Included |
| Account management and auth | — | Included |
| Provisioned agent container | — | Included |
| Data sync and backup | — | Included |
| Dashboard UI | — | Included |
| Support | Community | Included |

## How it works

When you sign up, the hosted product provisions a dedicated agent container with
your own data volume. Your farm state stays isolated. The web dashboard reads
synced data so you can check briefs, observations, and recommendations from a browser.

The core farm intelligence — tools, skills, knowledge, and recommendations — comes
from the same open-source agent kit documented on this site. The hosted product adds
the infrastructure, not the reasoning.

## Pricing

published pricing at launch.

## Self-hosting instead

If you prefer to run your own setup, the [Self-Hosting](/self-hosting) guide covers
everything you need to get started with SQLite, local scheduling, and your preferred
agent surface.
