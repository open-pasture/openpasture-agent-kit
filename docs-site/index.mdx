## The daily loop

Every day, the system helps answer four questions:

<Steps>
  <Step title="What is true about the farm right now?">
    Observations, weather, satellite data, and stored farm state assemble the current picture.
  </Step>
  <Step title="Should the herd MOVE, STAY, or does the agent NEEDS_INFO?">
    The morning brief produces a practical recommendation with reasoning.
  </Step>
  <Step title="Why does that recommendation make sense?">
    Every recommendation includes the data sources, knowledge entries, and assumptions behind it.
  </Step>
  <Step title="What single observation would reduce uncertainty most?">
    When the answer is not clear, the agent asks for the one thing that would change the call.
  </Step>
</Steps>

## Philosophy

- The farmer remains the operator. The agent extends attention, memory, and execution.
- The land is the feedback mechanism.
- Good grazing is contextual, not formulaic.
- Observation is scarce — direct it carefully.
- Leave room for uncertainty rather than pretending precision.
