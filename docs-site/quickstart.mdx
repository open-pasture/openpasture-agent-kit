# Quickstart

Get a local farm running in under five minutes.

## Requirements

- Python 3.11+
- An LLM provider key for your agent runtime
- Optional: Hermes for the Hermes adapter
- Optional: `FIRECRAWL_API_KEY` for external web knowledge ingestion

## Install

<Steps>
  <Step title="Install Hermes (optional)">
    Only needed if you plan to use the Hermes adapter.

    ```bash
    pip install "git+https://github.com/NousResearch/hermes-agent.git@1cec910b6a064d4e4821930be5cfaaf6145a2afd"
    ```
  </Step>
  <Step title="Install the agent kit">
    ```bash
    pip install -e .
    ```
  </Step>
  <Step title="Configure your environment">
    ```bash
    export OPENPASTURE_STORE=sqlite
    export OPENPASTURE_DATA_DIR="$HOME/.openpasture"
    export OPENPASTURE_BRIEF_TIME="06:00"
    ```

    Add `FIRECRAWL_API_KEY` only if you want external knowledge ingestion.
  </Step>
</Steps>

## First farm

List the available tools:

```bash
openpasture tools list
```

Create a simple farm:

```bash
openpasture tool run setup_initial_farm --json '{"name":"Willow Creek","timezone":"America/Chicago","herd":{"id":"herd_1","species":"cattle","count":28},"paddocks":[{"id":"paddock_home","name":"Home","status":"grazing"}]}'
```

Generate the first morning brief:

```bash
openpasture tool run generate_morning_brief --json '{}'
```

The brief returns a practical recommendation — `MOVE`, `STAY`, or `NEEDS_INFO` — with
reasoning visible enough for a farmer to judge.
