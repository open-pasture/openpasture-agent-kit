# Quickstart

Use this path when you want a local self-hosted test of the agent kit.

## Requirements

- Python 3.11+
- An LLM provider key for the agent runtime you use
- Optional: Hermes if you want the Hermes adapter
- Optional: `FIRECRAWL_API_KEY` for external web knowledge ingestion

## Install

From the agent kit repo:

```bash
pip install "git+https://github.com/NousResearch/hermes-agent.git@1cec910b6a064d4e4821930be5cfaaf6145a2afd"
pip install -e .
```

Recommended local settings:

```bash
export OPENPASTURE_STORE=sqlite
export OPENPASTURE_DATA_DIR="$HOME/.openpasture"
export OPENPASTURE_BRIEF_TIME="06:00"
```

Add `FIRECRAWL_API_KEY` only if you want the knowledge-ingestion workflow to
search and read outside sources.

## First Farm

List the available tools:

```bash
openpasture tools list
```

Create a simple farm:

```bash
openpasture tool run setup_initial_farm --json '{"name":"Willow Creek","timezone":"America/Chicago","herd":{"id":"herd_1","species":"cattle","count":28},"paddocks":[{"id":"paddock_home","name":"Home","status":"grazing"}]}'
```

Generate the first brief:

```bash
openpasture tool run generate_morning_brief --json '{}'
```

The brief should return a practical recommendation: `MOVE`, `STAY`, or
`NEEDS_INFO`, with the reasoning visible enough for a farmer to judge.
